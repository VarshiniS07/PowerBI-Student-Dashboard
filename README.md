# Student Performance Dashboard — Power BI

Interactive dashboard analyzing 100 students across 5 subjects (Grades 9–12).

## Key Findings
- Overall pass rate: 75.20% (pass mark: 60)
- Grade 12 pass rate: 92.50% vs Grade 9: 48.00%
- All subjects perform consistently (avg 68–70)
- Higher study hours strongly correlate with better scores

## Tools Used
Power BI Desktop, DAX, Excel

## Features
- KPI cards (avg score, max score, pass rate, attendance)
- Interactive slicers for Grade and Gender
- DAX measure for pass rate calculation
- 6 charts covering subject, grade, gender, and study hour analysis
